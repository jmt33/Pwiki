Pwiki
============

Pwiki develop version - 2014.8

by Mtao 

Introduction
------------

Pwiki 是一个简易的wiki框架，采用php语言编写

引用　 phpmarkdown

Quick Start
-----------


查看目前状态，有哪些列表
-----

```
php ./pwiki status 
```

创建一个新的wiki
-----

```
php ./pwiki new -t "title" -c "category"
```

生成html
-----

```
php ./pwiki generate [key] #key表示文件生成时候的时间
```
